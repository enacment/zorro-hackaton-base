# Bootstrap checklist (todos los equipos)

1. Fork del repo base provisto por Enacment
2. Configurar rama de equipo
3. Completar README del equipo
4. Validar secretos y entorno
5. Seleccionar proveedor de auth para la sesión (mock por default)
6. Ejecutar build baseline
7. Correr prueba mínima de integración + healthcheck
8. Registrar blockers (incluyendo pendientes de DLL)
