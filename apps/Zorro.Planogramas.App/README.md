# Zorro.Planogramas.App

## Arranque
1. Abrir `Zorro.sln` desde la raíz del repo.
2. Seleccionar este proyecto como startup project.
3. Llenar `context/PRD.md` con el PRD de tu equipo.
4. `dotnet build` (o build desde IDE) para validar.
5. Empezar Sprint 1.

## Estructura
- `src/Pages/` — tus pantallas
- `src/Auth/` — autenticación (MockAuthProvider activado por default)
- `src/Health/` — healthcheck de app y API
- `context/PRD.md` — fuente de verdad del agente
