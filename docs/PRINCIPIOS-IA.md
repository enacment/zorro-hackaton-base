# Principios de desarrollo asistido por IA

## Codex y Antigravity: rol esperado
- Codex: tareas de mayor complejidad técnica.
- Antigravity: aceleración de scaffolding, documentación y alternativas.

## Ciclo recomendado
1. Definir micro-objetivo
2. Prompt estructurado
3. Implementación
4. Build/test
5. Commit

## Prompt base
```txt
Contexto: [equipo] + [bloque PRD].
Stack: .NET MAUI + Azure.
Restricciones: sin features fuera de alcance.
Entrega: archivos, comandos, pruebas mínimas, riesgos.
```
