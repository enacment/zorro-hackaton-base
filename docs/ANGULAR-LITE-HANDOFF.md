# Handoff: versión Angular súper light (sin servidor)

## Objetivo
Tener un demo rápido para enseñar flujo de construcción sin levantar entorno .NET MAUI.

## Alcance del demo
- Frontend Angular standalone.
- Sin backend propio.
- Auth mock local (login/logout).
- Rutas mínimas: `login`, `home`, `health`.
- Healthcheck opcional a endpoint público de prueba.

## Qué se mantiene igual al template base
- Estructura de guardrails IA.
- PRD como única fuente de verdad.
- Iteraciones cortas con validación.

## Siguiente paso
Publicar repo Angular-lite separado para entrenamiento y uso en sesión.
