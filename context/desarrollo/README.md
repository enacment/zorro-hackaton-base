## Regla base para todos
Antes de cualquier prompt o implementación, carguen siempre:
1. `context/PRD.md`
2. `context/memory.md`
3. El contexto específico del caso (`context/planogramas/README.md`, `context/traspasos/README.md`, `context/precios/README.md`)

Si algo en el prompt contradice PRD, paramos y lo aclaramos primero.
